# Ayush Kiran - Portfolio Website

A modern, responsive portfolio website showcasing my projects, experience, education, and leadership roles.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations
- **Detailed Content**: Expanded descriptions of projects, internships, and leadership experiences
- **Easy Navigation**: Smooth scrolling navigation with mobile-friendly hamburger menu
- **Performance Optimized**: Lightweight and fast-loading

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- Vanilla JavaScript
- Google Fonts (Inter)

## Getting Started

1. Clone or download this repository
2. Open `index.html` in a web browser
3. That's it! No build process or dependencies required.

## File Structure

```
portfolio/
├── index.html      # Main HTML structure
├── styles.css      # All styling and responsive design
├── script.js       # Interactive features and animations
└── README.md       # This file
```

## Customization

To customize this portfolio:

1. **Content**: Edit `index.html` to update your information
2. **Styling**: Modify CSS variables in `styles.css` to change colors and themes
3. **Functionality**: Add or modify features in `script.js`
